# loconv

loconv is a lightweight local image converter that doesn't upload your data to any external servers
built using bun, vite, typescript and react

this project is viewable on cloudflare pages [here](https://loconv.pages.dev/)

## building/running loconv

### requirements

the rest of this section assumes you have bun and this project's dependencies installed

### development

to start the dev server

```bash
bun run dev
```

### production

to build the app for production

```bash
bun run build
```

## tech stack

- language: typescript
- frontend: react
- runtime: bun
- bundler: vite
- formater: biome
